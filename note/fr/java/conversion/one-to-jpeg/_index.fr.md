{{% blocks/products/pf/agp/feature-section-col title="Étapes pour convertir ONE en JPEG" %}}
1. Chargez UN fichier avec la classe Document
2. Créez un objet ImageSaveOptions avec SaveFormat.Jpeg comme paramètre.
3. Appelez la méthode Save avec le fichier image JPEG et enregistrez les options en tant que paramètres.
